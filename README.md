- C语言是古老而长青的编程语言，它具备了现代程序设计的基础要求，它的语法是很多其他编程语言的基础，在系统程序、嵌入式系统等领域依然是无可替代的编程语言。是后续的操作系统、编译原理、体系结构等课程的基石。
- 看懂一个C程序，和从零开始创造一个C程序是完全不同的两个概念。这就是为什么很多人自以为看懂了各种排序和查找算法，自己却绞尽脑汁也写不出来。
